# Alkesh James - Portfolio

Personal portfolio website for Alkesh James. Built with React, Vite, GSAP, and Lenis for smooth scrolling and animations.

## Features
- **Custom Cursor** with magnetic hover effects
- **GSAP Animations** for scroll-triggered elements and page reveals
- **Lenis Smooth Scrolling** for a premium feel
- **Responsive Typography** using PP Neue Montreal
- **Bento Grid** interactive layout

## Development
To run the project locally:
```bash
npm install
npm run dev
```

## Deployment
This project is connected to Vercel and automatically deploys whenever changes are pushed to the `main` branch.
